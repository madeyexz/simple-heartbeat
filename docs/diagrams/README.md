# Diagrams

All diagrams were created with [Excalidraw](https://excalidraw.com). Click any link below to open the interactive diagram, then export as PNG to update the README images.

## How to export PNGs

1. Click an Excalidraw link below
2. Select all elements (Cmd+A)
3. Export as PNG (Menu > Export image, or Cmd+Shift+E)
4. Save to this directory with the filename shown
